## 一、项目概览

基于云南地理特征的多灾种应急管理平台，覆盖灾情上报、AI方案生成、资源调度全流程。

**技术栈**: React 18 + FastAPI + PostgreSQL/PostGIS + DeepSeek AI + Milvus(RAG)

**团队**: 3人（后端开发 ×1 + 前端开发 ×1 + 产品/文档 ×1）
